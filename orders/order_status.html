Order Status List

Status Name Status Code Description
Inquiry inquiry UserB has submitted a request. The order is not yet created. For quote-based services, waiting for UserA to send a quote. For base-price services, UserA can either accept the order directly or adjust the quote based on the request.
Awaiting Payment awaiting_payment UserA has provided a quote, and UserB has accepted it, but payment has not yet been made. The final price is confirmed and awaiting payment to proceed.
In Progress in_progress UserB has completed payment. The order is officially created and now listed in the active orders. UserA starts working on the project.
Delivered delivered UserA has submitted the final deliverables. Awaiting confirmation from UserB.
In Revision in_revision UserB has requested revisions. The order remains open and will be updated accordingly.
Completed completed UserB has confirmed the work is complete, or the system has auto-completed the order after a set period.
Cancelled cancelled Either party has cancelled the order. This may occur before payment, during production, or after delivery.
